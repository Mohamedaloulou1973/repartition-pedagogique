# 📱 التوزيع البيداغوجي — معهد سبيطلة 2

تطبيق ويب تقدمي (PWA) للبحث في التوزيع البيداغوجي للسنة الدراسية 2025/2026.

## ✨ المميزات

- 🔍 بحث حسب الأستاذ أو القسم مع اقتراحات تلقائية
- 📴 يعمل بدون إنترنت بعد التثبيت
- 📱 قابل للتثبيت على الأيفون والأندرويد كتطبيق
- 🔤 واجهة عربية كاملة (RTL)
- ⚡ سريع وخفيف (لا خوادم ولا قواعد بيانات خارجية)

## 🚀 النشر على GitHub Pages

### الخطوة 1 — إنشاء المستودع
```bash
# أنشئ مستودعاً جديداً على GitHub باسم مثلاً:
# sbeitla2-distribution
# أو أي اسم تختاره
```

### الخطوة 2 — رفع الملفات

**الطريقة السريعة (عبر GitHub.com):**
1. افتح مستودعك على GitHub
2. اضغط **"Add file" → "Upload files"**
3. ارفع جميع الملفات مع الحفاظ على هيكل المجلدات:
   ```
   index.html
   manifest.json
   sw.js
   icons/
     icon-128.png
     icon-144.png
     icon-152.png
     icon-192.png
     icon-512.png
     apple-touch-icon.png
   ```
4. اضغط **"Commit changes"**

**الطريقة عبر Git (للمتقدمين):**
```bash
git init
git add .
git commit -m "Initial PWA release"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO_NAME.git
git push -u origin main
```

### الخطوة 3 — تفعيل GitHub Pages
1. اذهب إلى **Settings** في المستودع
2. من القائمة اليسرى اختر **Pages**
3. تحت **Source** اختر **"Deploy from a branch"**
4. اختر **Branch: main** ، **Folder: / (root)**
5. اضغط **Save**
6. انتظر دقيقة ثم افتح:
   ```
   https://USERNAME.github.io/REPO_NAME/
   ```

## 📱 التثبيت على الأيفون (iOS)

> **مهم:** يجب استخدام متصفح **Safari** فقط — Chrome وFirefox لا يدعمان تثبيت PWA على iOS

1. افتح الرابط في **Safari**:
   ```
   https://USERNAME.github.io/REPO_NAME/
   ```
2. اضغط على أيقونة المشاركة ⎙ (في الشريط السفلي)
3. مرر للأسفل واختر **"إضافة إلى الشاشة الرئيسية"**
4. اضغط **"إضافة"** في الزاوية العليا اليمنى
5. 🎉 سيظهر التطبيق على شاشتك الرئيسية!

---

## 🤖 التثبيت على Android

1. افتح الرابط في **Chrome**
2. سيظهر بانر تلقائي **"تثبيت التطبيق"** — اضغط عليه
3. أو: اضغط على القائمة ⋮ ثم **"إضافة إلى الشاشة الرئيسية"**

## 🔧 التحديث

لتحديث البيانات (إضافة أستاذ، تعديل قسم...):
1. افتح `index.html`
2. ابحث عن كائن `const DATA = {` (السطر ~120)
3. عدّل البيانات المطلوبة
4. ارفع الملف المعدَّل على GitHub
5. سيُحدَّث التطبيق تلقائياً على أجهزة المستخدمين
